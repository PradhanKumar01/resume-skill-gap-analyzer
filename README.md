# Resume Analyzer (Skill Gap Finder)

This is a Python script that analyzes a resume text and extracts relevant skills based on predefined job roles.  
It helps identify **which skills are already present** in a resume and **what skills are expected** for roles like **Data Analyst** and **Python Developer**.

This project is especially useful for **freshers**, **BCA students**, and **job seekers**.
